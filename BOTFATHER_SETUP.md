# BotFather MiniApp Configuration

## Для кнопки "Открыть МiniApp" в BotFather

### Правильная конфигурация:

1. **URL для кнопки MiniApp:**
   ```
   https://way.clans.idlebat.online/
   ```
   **БЕЗ параметров!** (не добавляй `?userId=...`)

2. **Почему это работает:**
   - Когда пользователь открывает MiniApp через кнопку в BotFather, Telegram передает информацию о пользователе через WebApp API
   - Фронтенд автоматически извлекает userId из `Telegram.WebApp.initDataUnsafe.user.id`
   - Это позволяет загружать данные Supabase правильно

3. **Почему важно БЕЗ параметров:**
   - Когда URL без параметров, Telegram применяет все настройки BotFather (borders, header color и т.д.)
   - Если в URL есть параметры, некоторые настройки BotFather могут не применяться корректно

## Для /start команды (опционально)

Текущий бот отправляет ссылку с параметром userId:
```
${process.env.MINIAPP_URL}?userId=${userId}
```

Это работает, но теперь не обязательно, так как фронтенд может получить userId из Telegram API.

## Проверка конфигурации

После установки правильного URL в BotFather:

1. Нажми /start команду
2. Нажми кнопку "🎮 Открыть МiniApp"
3. Проверь что:
   - Приложение загружается без ошибок
   - Видны твои данные из Supabase
   - Видны ресурсы вверху экрана
   - Видны кнопки навигации внизу экрана (даже если крутишь страницу)
   - Настройки BotFather применены (границы, цвета и т.д.)

## Если у тебя есть вопросы по настройке:

- Убедись что MINIAPP_URL переменная указывает на правильный домен
- Проверь что бэкенд запущен и доступен
- Посмотри консоль браузера на ошибки (F12 -> Console)
