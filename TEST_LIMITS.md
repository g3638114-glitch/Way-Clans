# 🧪 Тестовые случаи для казны и склада

## Тест 1: Лимит Склада при сборке ресурсов

**Цель**: Убедиться, что игрок не может собрать больше ресурсов, чем позволяет уровень склада

### Сценарий:
1. Создать нового игрока (по умолчанию уровень склада = 1, вместимость = 5000)
2. Произвести 2000 дерева в лесопилке (ниже лимита)
3. Собрать дерево ✅ (должно получиться 4500 дерева: 2500 стартовые + 2000)
4. Произвести ещё 2000 дерева
5. Попытаться собрать ✅ (получится только 500, чтобы не превысить 5000)
6. Попытаться собрать ещё раз ❌ (должна быть ошибка "Storage is full!")

### Ожидаемый результат:
```
Total wood = 5000 (лимит)
Сообщение об ошибке: "Storage is full! Can only collect X more wood"
```

## Тест 2: Лимит Казны при продаже

**Цель**: Убедиться, что игрок не может продать ресурсы, если казна переполнится

### Сценарий:

2. Попытаться продать дерево (100 шт = 1000 золота) ❌
3. Ошибка: "Treasury is full! Can only add X more gold"

### Ожидаемый результат:
```
Продажа блокирована
Золото = 5000 (не изменилось)
Дерево = 2500 (не изменилось)
```

## Тест 4: Лимит Казны при обмене

**Цель**: Убедиться, что обмен золота работает правильно и не влияет на лимит вместимости

### Сценарий:
1. Игрок с 5000 золота уровня 1 казны
2. Обмен: 1000000 золота на 1 жамкоин ❌ (недостаточно золота)
3. Улучшить казну на уровень 5 (вместимость = 80000)
4. Собрать золота до 1000000
5. Обмен: 1000000 золота на 1 жамкоин ✅
6. Должно остаться: 0 золота, 1 жамкоин

### Ожидаемый результат:
```
gold = 0
jabcoins = старое_значение + 1
```

## Тест 5: Апгрейд Казны

**Цель**: Проверить, что апгрейд казны увеличивает лимит

### Сценарий:
1. Уровень 1: вместимость = 5000

3. Проверить: вместимость = 10000
4. Попытаться улучшить до уровня 6 ✅ (должно работать, если уровней 6+)
5. Попытаться улучшить ещё раз ❌ (на максимальном уровне)

### Ожидаемый результат:
```
Уровень казны = максимальный уровень (5 или больше)
Вместимость = последнее значение в capacityPerLevel
Ошибка: "Treasury is already at maximum level"
```

## Как понять, работает ли всё правильно?

### ✅ Признаки успешной работы:
1. Не возможно собрать больше, чем позволяет склад
2. Не возможно заработать больше золота, чем позволяет казна
3. Апгрейды увеличивают вместимость
4. Фронтенд показывает правильные лимиты
5. Не теряются ресурсы при попытке собрать в переполненный склад

### ❌ Признаки проблем:
1. Ресурсы собираются больше, чем лимит
2. Золото добавляется больше, чем лимит казны
3. Мессаджи об ошибке не появляются
4. Ресурсы исчезают неожиданно
5. Фронтенд и бэкенд показывают разные лимиты
