# 🏰 Казна и Склад - Гайд по добавлению новых уровней

Это полное руководство по добавлению новых уровней для **Казны (Казна)** и **Склада (Склад)** в системе.

## ⚡ Быстрый старт (TL;DR)

1. Отредактируй **2 файла конфигурации** (сервер + клиент)
2. Добавь **одну строку** в массив `capacityPerLevel`
3. Добавь **одну строку** в объект `costs`
4. Готово! ✅

## 🔧 Как добавить уровень 7 для Казны

### Шаг 1️⃣: Отредактировать `src/config/buildings.js`

Найди `TREASURY_CONFIG` и добавь новую вместимость в конец массива:

```javascript
export const TREASURY_CONFIG = {
  name: 'Казна',
  capacityPerLevel: [
    31250, 
    62500, 
    125000, 
    250000, 
    500000, 
    1000000,
    2000000  // ← НОВЫЙ уровень 7
  ],
  costs: {
    2: { gold: 625, stone: 625, wood: 625 },
    3: { gold: 1250, stone: 1250, wood: 1250 },
    4: { gold: 2500, stone: 2500, wood: 2500 },
    5: { gold: 5000, stone: 5000, wood: 5000 },
    6: { gold: 10000, stone: 10000, wood: 10000 },
    7: { gold: 20000, stone: 20000, wood: 20000 }  // ← НОВАЯ стоимость уровня 7
  },
};
```

### Шаг 2️⃣: Отредактировать `public/js/game/config.js`

Найди `TREASURY_CONFIG` в **этом же файле** и сделай те же изменения:

```javascript
export const TREASURY_CONFIG = {
  name: 'Казна',
  icon: '🏰',
  resource: 'gold',
  capacityPerLevel: [
    31250, 
    62500, 
    125000, 
    250000, 
    500000, 
    1000000,
    2000000  // ← НОВЫЙ уровень 7
  ],
  costs: {
    2: { gold: 625, stone: 625, wood: 625 },
    3: { gold: 1250, stone: 1250, wood: 1250 },
    4: { gold: 2500, stone: 2500, wood: 2500 },
    5: { gold: 5000, stone: 5000, wood: 5000 },
    6: { gold: 10000, stone: 10000, wood: 10000 },
    7: { gold: 20000, stone: 20000, wood: 20000 }  // ← НОВАЯ стоимость уровня 7
  },
};
```

### Готово! ✅

Больше ничего менять не нужно. Система автоматически:
- ✅ Обновит максимум уровней (с 6 на 7)
- ✅ Проверит лимиты при сборе ресурсов
- ✅ Покажет новую вместимость в UI
- ✅ Позволит игроку улучшить казну до уровня 7

## 🔐 Как работает система валидации

**Казна (Treasury)** - Золото не может превышать лимит:
- `addGold()` в `src/services/resourceService.js` проверяет максимум казны перед добавлением
- `sellResources()` проверяет что золото от продажи не превысит лимит
- `collectResources()` проверяет склад перед сбором

**Склад (Storage)** - Ресурсы не могут превышать лимит:
- `collectResources()` в `src/services/buildingService.js` проверяет вместимость перед сбором
- `sellResources()` отнимает ресурсы со склада
- Каждый ресурс (дерево/камень/мясо) имеет одинаковый лимит

**Все проверки** выполняются на сервере (безопасно)! ✅

```javascript
// Автоматически считает из длины массива!
function getTreasuryMaxLevel() {
  return TREASURY_CONFIG.capacityPerLevel.length;
}
// Если в массиве 7 элементов → максимум уровень 7
```

### Как проверяются лимиты?
```javascript
// Сервер проверяет перед любым добавлением ресурса:
const treasuryLevel = user.treasury_level || 1;
const treasuryCapacity = getTreasuryCapacity(treasuryLevel);
const newGoldAmount = user.gold + goldEarned;

if (newGoldAmount > treasuryCapacity) {
  throw new Error(`Treasury is full!`);
}
```

## ❌ Частые ошибки

### ❌ Неправильно: Добавить уровень только в один конфиг
```javascript
// НЕПРАВИЛЬНО! ❌
// Отредактировал только src/config/buildings.js
// А public/js/game/config.js забыл обновить
```

**ПРАВИЛЬНО:** Всегда отредактируй **ОБА файла** (сервер + клиент)

### ❌ Неправильно: Забыть добавить стоимость
```javascript
// НЕПРАВИЛЬНО! ❌
capacityPerLevel: [31250, 62500, 125000, 250000, 500000, 1000000, 2000000],
// Забыли добавить costs[7]!
```

**ПРАВИЛЬНО:** Добавь новый уровень в массив **И** в объект стоимостей

### ❌ Неправильно: Отрицательные или нулевые значения
```javascript
// НЕПРАВИЛЬНО! ❌
capacityPerLevel: [31250, 0, 125000, ...] // нулевая вместимость?
costs: { 2: { gold: -100, ... } } // отрицательная стоимость?
```

## 🧪 Как проверить что всё работает

1. **Добавь новый уровень** (7-й) в оба конфига
2. **Открой игру** в браузере (разработчик)
3. **Проверь консоль** - нет ли ошибок?
4. **Попробуй улучшить казну/склад** - работает ли новый уровень?
5. **Собери ресурсы** - проверяются ли лимиты?

## 📞 Вопросы?

Если что-то не работает:
1. Проверь что отредактировал **оба файла** (сервер + клиент)
2. Проверь что **стоимость есть** для каждого нового уровня
3. Проверь что **нет опечаток** в названиях (capacityPerLevel, costs)
4. Перезагрузи страницу браузера (Ctrl+Shift+R)
